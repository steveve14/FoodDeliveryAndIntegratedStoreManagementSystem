## 🔄 P1 — 진행 중 / 예정

### P1-1. 사장님 대시보드 핵심 기능

- [ ] 통합 주문 목록 (배달/매장)
- [ ] 기본 매출 지표
- [ ] 메뉴 품절 토글 (실시간 반영)
- **완료 조건**: 운영자 관점 핵심 화면 3종 연동 완료

### P1-2. 실시간 주문 알림

- [ ] 사장님 → 새 주문 알림 수신
- [ ] 고객 → 주문 상태 변경 알림
- **기술 옵션**: WebSocket 또는 SSE (Server-Sent Events)

### P1-3. 결제 연동

- [ ] PG사(아임포트/토스페이먼츠) 연동
- [ ] 결제 완료 시 주문 상태 자동 전이
- [ ] 영수증 발행

### P1-4. Android 앱 API 완전 연동

- [ ] `app-android-shop`: 사장님 주문 관리 완전 연동
- [ ] `app-android-user`: 고객 주문/결제 완전 연동
- [ ] `app-android-kiosk`: 매장 테이블 오더 완전 연동
- [ ] `app-android-delivery`: 배달 배차/완료 완전 연동

### P1-5. 매장 레이아웃 (드래그 앤 드롭)

- [ ] `web-shop` 테이블 배치도 드래그 앤 드롭 구현
- [ ] 테이블별 주문 현황 실시간 표시

- [ ] 서비스 간 이벤트 기반 통신으로 전환
- [ ] 주문 생성 이벤트 → 배달/알림 서비스 비동기 처리
- [ ] 상세 계획: `docs/예정/Kafka_도입_계획.md` 참조

### P2-2. Kubernetes 배포 자동화

- [ ] 서비스별 K8s Deployment / Service YAML 작성
