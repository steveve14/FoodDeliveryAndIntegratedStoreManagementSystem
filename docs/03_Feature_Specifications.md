# 03. 상세 기능 명세서 (Feature Specifications)

## 1. 공통 기능 (Auth Service)
*   **회원가입/로그인**: 이메일, 소셜 로그인 (OAuth2), JWT 발급.
*   **권한 관리**: 고객(Customer), 사장님(Owner), 직원(Staff), 라이더(Rider), 관리자(Admin) 권한 분리.

## 2. 매장 시스템 (Store Service + Tablet Front)
*   **테이블 오더**: 고객이 태블릿으로 메뉴 확인 및 장바구니 담기. (결제 없이 주방 전송 가능)
*   **직원 호출**: 물, 반찬, 숟가락 등 요청사항 선택 호출.
*   **POS 기능**: 테이블별 주문 내역 합산, 후불 결제 처리, 영수증 출력 연동.

## 3. 사장님 대시보드 (Store Service + Order Service + Web Front)
*   **매장 레이아웃 설정**: 드래그 앤 드롭으로 실제 매장 테이블 배치도 구현.
*   **통합 주문 접수**: [배달] 주문과 [매장] 주문을 하나의 리스트에서 확인.
*   **메뉴/재고 관리**: 메뉴 품절 처리 시 태블릿과 배달 앱에 실시간 반영.
*   **매출 분석**: 일별/월별 매출 차트 제공 (배달 매출 vs 매장 매출 비교).

## 4. 고객 배달 앱 (Order Service + Mobile App)
*   **주문 프로세스**: 가게 검색 -> 메뉴 선택 -> 장바구니 -> 결제 -> 주문 접수.
*   **실시간 상태**: 접수 대기 -> 조리 중 -> 배달 중 -> 배달 완료 상태 푸시 알림.

## 5. 라이더 앱 (Rider Service + Mobile App)
*   **배차 시스템**: 근처 배달 요청 콜 리스트 확인 및 수락.
*   **배달 처리**: 픽업 완료 및 배달 완료 처리.
